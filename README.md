# IS436-Group5
Implement an online chat application (using socket programming) that supports at least two users (one server and one client) to chat remotely. 

The application will consist of a Chat Server and Chat client that will be console-based and programed using Python. 
Extra feature will be added down the line.

Chat Server Requirements
--------------------------
-Listen on specified TCP port

-Verify that TCP is within range

-Handles one connection or more

-Communication is supported

-Successful Termination


Chat Client Requirements
-------------------------
-Recieves specified TCP connection

-Can sustain communication with server

-End connection with server upon receiving a command

-Succesful Termination
